# Music library

Drop **cleared** audio here — tracks you have the rights to post (your own
originals, royalty-free / licensed library music, or tracks the rights holder
approved).

The agent only uses files from this folder for music beds. It will never rip
trending or copyrighted audio into a render, because API-posted videos with
unlicensed music risk being muted or taken down.

Suggested naming: `<vibe>-<name>.mp3`, e.g. `hypnotic-loop.mp3`,
`euphoric-closer.mp3` — the vibe prefix helps the agent match tracks to a
clip's mood (see the `music:` fields in `accounts/accounts.example.yaml`).

Audio files are gitignored (`assets/music/*.mp3`) so your library never gets
committed.
